HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • An "H 1" header is the top priority followed be H 2,3,4,5, and so on.
  • "p" stands for paragraph, typically used for text.
  • "b" stands for break, giving space between elements on a website.

CSS

  • In CSS we use class selectors, like ".card" which is used to style topics.
  • Padding is commonly confused with margin, but they are not the same.
  • Padding is an "interal" in regards to its element, where a margin is "external".
  • Increasing the padding makes the element itself larger.
  • "*" is known as a "special wild card symbol" which applies to all elements, not a specified style class.

Git

  • Every command in Git bash begins with "git"
  • Using git is so far my favorite part of coding, I enjoy file directory commands/exploration.
  • "pwd" stands for "print working directory" effectively telling you where you are within git.
  • "ls" lists the current items within the directory you are currently in.

JavaScript

  • Control flow is the order in which a computer executes code in a script, being specific and mindful of the order of code is crucial.
  • Calling the function is as equally important as is defining the function.
  • The first index (an individual variable) of an array (a group of related data) is always 0.
  • Javascript is the most difficult portion of coding so far, but I think I'll come to like it. Or at least appreciate it.
